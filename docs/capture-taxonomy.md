# Capture Taxonomy

Use these topics when classifying raw captures into generated Starlight summary pages.

- `ai-coding`: AI-assisted development tools, workflows, prompts, and editor agents.
- `documentation`: Docs strategy, writing systems, site structure, and knowledge-base maintenance.
- `github`: GitHub issues, pull requests, Actions, security, and repository workflows.
- `codex`: Codex-specific cloud, CLI, app, automation, or integration behavior.
- `cloud-compute`: General cloud hosting, runtime, serverless, and compute infrastructure.
- `vercel`: Vercel deployment, hosting, routing, and build behavior.
- `cloudflare`: Cloudflare DNS, Workers, Pages, security, and edge services.
- `azure`: Azure platform services, identity, Functions, storage, and app hosting.
- `fabric`: Microsoft Fabric platform, lakehouse, data engineering, and notebooks.
- `power-bi`: Power BI modeling, reports, visuals, administration, and data sources.
- `windows`: Windows setup, WSL, Store, shell, and workstation configuration.
- `devops`: CI/CD, release, environments, infrastructure-as-code, and operational practice.
- `reference`: Durable reference material that is useful but not tied to a current project.
- `research`: Exploratory material, comparisons, early signals, and ideas needing validation.
- `personal-workflow`: Personal productivity, capture, review, and automation systems.
